2. Теги, Атрибуты, Селекторы, Свойства

 1 - Тег
(в контексте CSS) — имя HTML элемента (например, p, div, h1), которое можно использовать как селектор.
Зачем нужно? Позволяет применять стили ко всем элементам этого типа на странице.

 2 - Атрибут
Дополнительная информация внутри HTML тега (например, class="header" или id="main").
Зачем нужно? Служит для точной выборки элементов в CSS (через селекторы по классу или ID).

 3 - Селектор
«Адрес», по которому CSS находит нужные HTML‑элементы.
Зачем нужно? Определяет, к каким элементам применить стили.
 - Могут быть по тегу (p, div), классу (.class), ID (#id), атрибуту ([href]) и др.;
 - Несколько селекторов через запятую: .btn, .link { ... };
Примеры:
  - p — все абзацы;
  - .highlight — элементы с class="highlight";
  - #unique — элемент с id="unique";
  - div p — абзацы внутри <div>.
Структура записи селектора:
селектор {
 свойство: значение;
 свойство: значение;
}

 4 - Свойство
Параметр стиля, который вы задаёте (например, color, font-size, margin).
Зачем нужно? Управляет внешним видом элемента (цвет, размер, отступы и т. д.).
Структура записи свойства:
 - Свойство и Значение разделяются двоеточием: color: red;
 - Каждое свойство заканчивается точкой с запятой (можно опустить у последнего);
 - Значения без единиц (например, 0) не требуют указания единиц измерения.

Выравнивание изображений с атрибутом align, его аналог (text-align, vertical-align)

- align — выравнивание, устарел;
- text-align — горизонтальное выравнивание содержимого блока;
- vertical-align — вертикальное выравнивание элемента внутри строки/ячейки.

  align - HTML атрибут для тега <img>, задавал горизонтальное выравнивание изображения относительно текста.
Основные значения:
 - left — изображение слева, текст обтекает справа;
 - right — изображение справа, текст обтекает слева;
 - top — верх изображения выравнивается с верхней частью строки;
 - middle — центр изображения по центру строки;
 - bottom — низ изображения по нижнему краю строки (по умолчанию).
Важно:
 - атрибут align устарел в HTML5;
 - вместо него используют CSS.

 text-align (CSS аналог для горизонтального выравнивания)
Что делает: выравнивает содержимое блока по горизонтали.
Где применяется: к блочным элементам (<div>, <p>, заголовки и т. п.).
Значения:
 - left — по левому краю;
 - right — по правому краю;
 - center — по центру;
 - justify — по ширине (растягивает строки).

 vertical-align (CSS аналог для вертикального выравнивания)
Что делает: выравнивает элемент по вертикали внутри строки или ячейки таблицы.
Где применяется: к строчным (<span>, <img>) и табличным элементам.
Значения:
 - baseline — по базовой линии текста (по умолчанию);
 - top — верх элемента по верхнему краю строки;
 - middle — центр элемента по центру строки;
 - bottom — низ элемента по нижнему краю строки;
 - text-top — верх по верхнему краю текста;
 - text-bottom — низ по нижнему краю текста.

Отличия:
 - text-align — для горизонтального выравнивания содержимого блока (текст, изображения внутри блока).
 - vertical-align — для вертикального выравнивания элемента внутри строки (например, картинка среди текста) или ячейки таблицы.
 - Не путайте: vertical-align не работает для блочных элементов в потоке — только для строчных и табличных.

Свойство background(-color, -image, -repeat, -position)

1. background-color
 Задаёт цвет фона элемента (можно использовать названия цветов, HEX, RGB, RGBA).
 Пример: background-color: #ff0000; (красный).
2. background-image
 Устанавливает изображение в качестве фона (путь к файлу через url()).
 Пример: background-image: url(image.jpg);
 Чтобы убрать фон: background-image: none;.
3. background-repeat
Определяет, как повторять фоновое изображение:
 - repeat — по горизонтали и вертикали (по умолчанию);
 - repeat-x — только по горизонтали;
 - repeat-y — только по вертикали;
 - no-repeat — не повторять (одно изображение).
4. background-position
Указывает, где разместить фоновое изображение внутри элемента.
Можно задавать:
 - ключевые слова: top, bottom, left, right, center;
 - значения в пикселях/процентах: 50% 50%, 10px 20px.
 Пример: background-position: center top; (по центру вверху).
5. background-size
Указывает на управление размерами фонового изображения.
Можно задавать:
 - ключевые слова: auto, cover, contain ;
 - значения в пикселях/процентах: ширина / ширина-высота, px, em, % и др.
 Пример: background-size: 100% 100%.

Свойство position

Определяет, как элемент располагается на странице относительно других элементов или окна браузера.
Основные значения: static, relative, absolute, fixed, sticky.

Свойство display

 «Режим отображения» элемента. От него зависит, как элемент ведёт себя в потоке страницы.
Определяет, как элемент будет отображаться на странице: его тип (блочный, строчный и др.) и поведение в потоке документа.
Ключевые значения:
1. block
 - Занимает всю доступную ширину.
 - Начинается с новой строки.
 - Можно задать ширину/высоту.
 - Примеры: div, p, h1.
2. inline
 - Ширина равна содержимому.
 - Располагается в строке с другими элементами.
 - Ширина/высота не работают.
 - Примеры: span, a, strong.
3. inline-block
 - Как inline, но можно задать ширину/высоту.
 - Не переносится на новую строку (если хватает места).
4. onen
 - Элементполностью скры (не занимает места).
 - Пример: для скрытия блоков через JS.
5. flex
 - Делает элемент flex контейнером (гибкое выравнивание дочерних элементов).
6. grid
 - Превращает элемент в grid контейнер (сетка для сложной вёрстки).

Границы: свойства border

Задаёт рамку вокруг элемента.
 border: ширина стиль цвет
 border: 1px solid black;
Отдельные свойства:
 - border-width — ширина (например, 1px, 2px);
 - border-style — стиль (solid, dashed, dotted, none, double и др.);
 - border-color — цвет (название, HEX, RGB).
 - border-radius — скругление углов (единицы: px, %);
Для отдельных сторон:
 - border-top, -right, -bottom, -left.

Полупрозрачность: свойство opacity

CSS-свойство, задающее прозрачность элемента.
Значения: число от 0 (полностью прозрачен) до 1 (непрозрачен).
Пример:
 - opacity: 1; — элемент полностью виден;
 - opacity: 0.5; — полупрозрачный (50% прозрачности);
 - opacity: 0; — невидимый.
Важно:
 - Применяется ко всему элементу (включая текст и дочерние элементы);
 - Не влияет на доступность (элемент остаётся в потоке документа).

Псевдоклассы ссылок(hover, link)

1. a:link
 - Применяется к непосещённым ссылкам (обычное состояние).
 - Пример: a:link { color: blue; }.
2. a:hover
 - Срабатывает, когда курсор наведён на ссылку.
 - Используется для визуальной обратной связи (например, смена цвета/подчёркивания).
 - Пример: a:hover { color: red; text-decoration: underline; }.
Важно:
 - Порядок в CSS имеет значение: обычно пишут a:link, затем a:hover.
 - :hover не работает на мобильных устройствах (нет курсора).
 - Для плавных эффектов можно добавить transition

3. Списки

Создание списков: упорядоченные ol li, неупорядоченные ul li.

 <ol> - Нумерованный - формируется с помощью тега <ol> - ordered list(сам лист) и <li> - list item (пункты списка)
 <ul> - Маркированный - (unordered list) - <ul> + <li> - имеет маркеры.

Форматирование списков: Свойства(list-style-type, -image, -position)

Свойства стилизации списков:
1. list-style-type
 Что делает: задаёт *вид маркера* списка.
 Пример: значений:
  - ul: disc (точка), circle (пустой кружок), square (квадрат);
  - ol: decimal (1, 2, 3…), lower-alpha (a, b, c…), upper-roman (I, II, III…);
  - none — убирает маркеры.
 По умолчанию: disc для ul, decimal для ol.
2. list-style-image
 Что делает: ставит изображение вместо стандартного маркера.
 Значения: url("путь/к/изображению.png") или none (стандартный маркер).
 Важно: размер изображения лучше подобрать заранее — CSS не масштабирует маркер автоматически.
3. list-style-position
 Что делает: определяет положение маркера относительно текста.
 Значения:
  - outside — маркер слева от текста (по умолчанию);
  - inside — маркер внутри текстового блока, участвует в обтекании.

Списки определений dl dd dt

 Это HTML-теги для разметки пар «термин — определение» (как в словаре или глоссарии).
Как работают теги:
 <dl> (definition list) — контейнер списка (обёртка для всех пар).
 <dt> (definition term) — термин (что объясняем).
 <dd> (definition description) — описание/определение термина.

4. Отступы и Поля

Свойство margin: margin-left, -top, right, -bottom.

 margin - внешний отступ: сверху справа снизу слева.

Свойство padding: padding-left, -top, right, -bottom.

 padding - внутренний отступ: сверху справа снизу слева.
 padding: 10px 20px; - первое по оси Х, второе по оси У
